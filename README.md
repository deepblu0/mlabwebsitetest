# M-LAB Landing Page

İzmir Mission Action Lab (M-LAB) için geliştirilmiş modern, etkileşimli ve responsive açılış sayfası.

## Proje Hakkında

Bu proje, İzmir'in iklim nötr ve akıllı şehir hedeflerini tanıtan, yürütülen projeleri ve paydaşları sergileyen bir web uygulamasıdır. Kullanıcı dostu arayüzü ve dinamik içerik yönetimi ile ziyaretçilere kapsamlı bir deneyim sunar.

## Kullanılan Teknolojiler

-   **React**: Kullanıcı arayüzü ve bileşen yönetimi için.
-   **Tailwind CSS**: Hızlı ve modern stillendirme için.
-   **Vite**: Hızlı geliştirme ve build süreçleri için.
-   **Lucide React**: Modern ikon seti için.

## Kurulum Komutları

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

1.  Projeyi klonlayın:
    ```bash
    git clone https://github.com/kullaniciadi/m-lab-landing.git
    cd m-lab-landing
    ```

2.  Bağımlılıkları yükleyin:
    ```bash
    npm install
    ```

3.  Geliştirme sunucusunu başlatın:
    ```bash
    npm run dev
    ```

4.  Production build almak için:
    ```bash
    npm run build
    ```

## Özellikler

-   **Dinamik Proje Filtreleme**: Projeleri kategorilerine (Enerji, Ulaşım, vb.) göre filtreleyin.
-   **Etkileşimli Harita**: İzmir haritası üzerinde proje konumlarını ve detaylarını görüntüleyin.
-   **Responsive Tasarım**: Mobil ve masaüstü cihazlarla tam uyumlu.
-   **Modern UI**: Glassmorphism efektleri ve yumuşak geçişler.
